# DiaScript
SNUGDC 스타터 프로젝트

키보드 궤적으로 마법을 시전하는 탑다운 액션 게임 (현재 프로토타입).

## 핵심 아이디어
키패드(Y U I O P / H J K L ; / N M , . /)의 키를 순서대로 눌러
궤적의 형태(V, Λ, O, X 등)를 인식하고 그에 맞는 마법을 발동.
붉은 마룡(공격)과 푸른 마룡(제어)을 전환하며 같은 형태로 다른 효과를 냄.

## 플레이 방법
| 키 | 동작 |
|---|---|
| WASD | 이동 |
| Shift | 대쉬 (무적) |
| Q / E | 붉은/푸른 마룡 전환 |
| 스펠키 입력 → Enter | 마법 발동 |
| 마지막 키 재입력 | 마법 발동 |
| 첫 번째 키 재입력 | 입력 취소 |
| Esc | 입력 취소 |

## 구현된 마법 형태
| 형태 | 인식 조건 | 붉은 효과 | 푸른 효과 |
|---|---|---|---|
| V | 양 끝 같은 층, 중간이 낮은 층 | 폭발 수렴 (꼭짓점으로 당김+피해) | 기절 수렴 |
| Λ | 양 끝 같은 층, 중간이 높은 층 | Λ 폭발 수렴 | Λ 기절 수렴 |
| O | 3→2→1→2층 순환 | 슬로우+지속 피해 | 봉인+지속 피해 |
| X | 3층 2개→2층 1개→1층 2개, 중간이 사이 | 교차선 피해 | 경로 전환+기절 |
| 기타 | 위 외 모든 입력 | 공격선 | 방어벽 |

## 구현된 몬스터
- **멧돼지**: idle→준비(타원)→돌진(삼각형) 사이클, V/O로 돌진 취소
- **거미**: 소환 위치에 십자 거미줄 고정, 접촉 시 슬로우
- **부유정령**: 랜덤 부유, 주기적 투사체 발사
- **분열슬라임**: 사망 1.5초 후 2분열 (붉은O=4분열, 푸른O=분열 없음)
- **뿔개미**: 2마리가 선으로 연결, I/X 용언으로 분리
- **폭탄벌레**: 사망 시 범위 폭발

## 기술 스택
- Vanilla JS + HTML5 Canvas
- 외부 라이브러리 없음
- 수학적 패턴 인식 (키 층/열 좌표 기반)
